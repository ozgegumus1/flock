# Flock 🎵

Upbeat, React ve TypeScript ile geliştirilmiş modern bir sosyal medya uygulamasıdır.

## 🌐 Canlı Demo

[upbeat-two.vercel.app](https://upbeat-two.vercel.app)

## ✨ Özellikler

- 🔐 Kullanıcı kaydı ve girişi
- 📝 Post paylaşma ve beğenme
- 👤 Profil sayfası ve düzenleme
- 👥 Takip et / Takipten çık
- 🔔 Bildirim sistemi
- 📱 Mobil uyumlu tasarım

## 🛠️ Kullanılan Teknolojiler

- **React** — Kullanıcı arayüzü
- **TypeScript** — Tip güvenliği
- **Tailwind CSS** — Stil
- **Supabase** — Veritabanı ve kimlik doğrulama
- **React Router** — Sayfa yönlendirme
- **Vite** — Geliştirme ortamı
- **Vercel** — Yayınlama
